# @botc/grimoire — цифровой гримуар (заглушка)

SPA-приложение ведущего. Пока не реализовано — только каркас пакета.

## Планируемый стек

- **Vite + React + TypeScript** — быстрая разработка, hot reload.
- **@botc/core** — переиспользуем модель игры (роли, статусы, типы).
- **Zustand / Redux Toolkit** — для централизованного `GameState`.
- **LocalStorage / IndexedDB** — для сохранения активной партии и истории.
- **TailwindCSS** — быстрый стайлинг в духе текущих `sheet.html` / `jinxes.html`.

## MVP scope

1. Рассадка 5–15 игроков по кругу (drag-n-drop seat).
2. Раздача ролей: автоматическая по составу или ручная.
3. Подсказки по ночному порядку: кто сейчас просыпается, что показать.
4. Трекинг статусов: отравлен, защищён, метка Векны, переманен.
5. Лог событий: день/ночь, кто что сделал, кто умер.
6. Экспорт `protocol.json` для ретроспективного разбора.

## Что не входит в MVP

- Сетевой режим — это будет `@botc/server` + его клиентская часть.
- Автоматические роллы информации (пусть ведущий решает, что показать).
- Мультиплеерные подключения игроков.
